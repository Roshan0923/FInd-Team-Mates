Find Team mates application will help the user to find the team mates and start a joureny of creating an exting project.The user can also share their project ideas and other user can join the project.So This platform helps the user to find team mates from around the world and connect with them.

Technical Details

This web application is built using Angular version 9.1.7 and Spring boot version 2.3.0 .I have used Angular for my front-end part and spring boot for my back-end part.Front-end part communicate with the back-end part using API. And each timeJWT token is being passed with the request.
